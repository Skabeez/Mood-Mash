## âš¡ Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Set Up API Keys
Create `.env` file (see [API_SETUP_GUIDE.md](API_SETUP_GUIDE.md)):
```env
DEEPSEEK_API_KEY=your_key
LASTFM_API_KEY=your_key
YOUTUBE_API_KEY=your_key
SUPABASE_URL=your_url
SUPABASE_ANON_KEY=your_key
```

### 3. Run the App
```bash
npm start
# Press 'i' for iOS or 'a' for Android
```

### 4. Test It
Send a message: "I want upbeat workout music" and watch the AI generate recommendations!
